# A股板块资金流向 · 动态桑基图

每个交易日自动抓取东方财富概念+行业板块主力资金净流入数据，渲染为 18 秒抖音竖屏（1080×1920）动态桑基图视频，并通过邮件发送到你的邮箱。

---

## 产品效果

**两段视频，每天自动产出：**

| 时段 | 数据窗口 | 收到时间 |
|------|---------|---------|
| 午盘 | 09:30–11:30 | 约 11:32 |
| 收盘 | 全天 09:30–15:00（午休跳过，动画连续） | 约 15:03 |

**画面内容：**

- 顶部标题「X月X日 午盘/收盘 资金流向」
- 右上角实时交易时钟
- 三列桑基图：左列净流出（绿）→ 中枢「主力资金」→ 右列净流入（红）
- 29 个固定追踪板块，按资金方向自动分配左右
- 差额由「主力抢筹」（金）/「主力跑路」（灰）节点配平，桑基永远闭合
- 顶部全市场氛围条：涨跌家数 + 成交额 + 较昨量变
- 底部水印「@主力去哪了」

**视觉风格：** 深蓝近黑科技风背景 + 细网格 + 霓虹辉光 + 缎带水平渐变。

## 运行方式（本地电脑）

### 首次配置

```powershell
cd D:\a0_Reasonix\a0_Sankey\local
pip install -r requirements.txt
```

设置邮箱环境变量（QQ 邮箱需使用授权码，非登录密码）：

```powershell
set MAIL_USERNAME=你的QQ邮箱@qq.com
set MAIL_PASSWORD=你的16位授权码
```

### 测试

```powershell
python main.py --once          # 抓一次真实数据确认接口通
python preview.py              # 生成一张静态桑基图预览
```

### 交易日自动运行

```powershell
python main.py                 # 全天自动：采集→渲染→发邮件
```

脚本会在 09:30 开始采集，11:30 渲染午盘并发送邮件，13:00 继续采集，15:00 渲染收盘并发送邮件。

### Windows 计划任务（全自动）

1. `Win+R` → `taskschd.msc`
2. 创建任务 → 触发器：每天 09:25
3. 操作：程序 `python`，参数 `main.py`，起始于 `D:\a0_Reasonix\a0_Sankey\local`

## 可调参数

所有参数集中在 `config.py`：

| 参数 | 默认值 | 说明 |
|------|--------|------|
| `DURATION` / `FPS` | 18s / 30fps | 视频时长和帧率 |
| `SAMPLE_INTERVAL_MIN` | 2 | 采集间隔（分钟） |
| `MIN_BAND_PX` | 3 | 最小条带高度（像素） |
| `node_gap` / `node_gap_min` / `node_gap_few` | 13/17/25 | 板块间隙（智能分级） |
| `stack_top` / `stack_bottom` | 300/1720 | 桑基图纵向范围 |
| `CRF` | 20 | 视频质量（越小越清晰） |
| `SHOW_MARKET_BAR` | True | 是否显示顶部市场氛围条 |
| `TRACKED_CONCEPT` / `TRACKED_INDUSTRY` | — | 固定追踪板块清单 |

## 数据源

优先东方财富主力净流入（口径准），东财不可达时自动回退同花顺净额。单次采集失败最多重试 3 轮，不轻易放弃。
